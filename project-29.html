Weddingz - Mobile App

Weddingz is a one stop shop for all wedding services like venue, make-up artist, photographer etc. They already had web presence and wanted to extend it on apps. This project mainly involved working on the user flow and app architecture through low-fidelity wireframes. The project was challenging as it required an extensive range of wedding services and products to be part of one single app. It was quite interesting to work out the flow to make it easy to use without making the user feel overwhelmed by the thousands of options.
